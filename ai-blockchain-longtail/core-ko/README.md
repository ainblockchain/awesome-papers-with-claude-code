# AI, Blockchain & The Long Tail

Claude Code 기반 인터랙티브 학습 경로로,
AI와 블록체인 기술의 교차점을 탐구합니다.

## 시작하기

1. 이 디렉토리에서 Claude Code를 엽니다:
   cd ai-blockchain-longtail/core-ko/
   claude
2. 자연스럽게 대화하며 학습을 시작하세요:
   explore              # knowledge graph 보기
   teach me <concept>   # 레슨 시작
   give me a challenge  # 퀴즈 풀기
   done                 # 완료 표시, 다음으로 이동

## 코스 구조

- **AI 중앙화 문제** (4개 개념): 왜 AI가 거대 기술 기업에 의해 지배되고 민주화를 방해하는 근본적인 도전을 이해합니다
- **솔루션 인프라로서의 블록체인** (4개 개념): 탈중앙화 컴퓨팅 플랫폼이 신뢰 없는 시장과 프로그래밍 가능한 인센티브를 통해 AI 개발을 위한 새로운 모델을 가능하게 하는 방법
- **자율 AI 시스템** (5개 개념): 자율 마켓플레이스, 품질 메커니즘, 탈중앙화 조정을 통해 자체 지속 가능한 커뮤니티 소유 AI 구축
- **인프라 진화 & 미래** (5개 개념): 인프라 투자 퍼즐, AINFT 비전, 조합성 시대, Layer 2 스케일링 진화 이해
