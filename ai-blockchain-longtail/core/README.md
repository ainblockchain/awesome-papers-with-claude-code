# AI, Blockchain & The Long Tail Learning Path

A Claude Code-powered interactive learning path exploring
the intersection of AI and blockchain technology.

## Getting Started

1. Open Claude Code in this directory:
   cd ai-blockchain-longtail/core/
   claude
2. Start learning -- just chat naturally:
   explore              # see the knowledge graph
   teach me <concept>   # start a lesson
   give me a challenge  # get a quiz
   done                 # mark complete, move on

## Course Structure

- **The AI Centralization Problem** (4 concepts): Understanding why AI has become dominated by big tech and the fundamental challenges preventing democratization
- **Blockchain as Solution Infrastructure** (4 concepts): How decentralized computing platforms enable new models for AI development through trustless markets and programmable incentives
- **Autonomous AI Systems** (5 concepts): Building self-sustaining, community-owned AI through autonomous marketplaces, quality mechanisms, and decentralized coordination
- **Infrastructure Evolution & the Future** (5 concepts): Understanding the infrastructure investment puzzle, AINFT vision, composability eras, and Layer 2 scaling evolution
