# Attention Is All You Need Learning Path

A Claude Code-powered interactive learning path based on
"Attention Is All You Need" by Vaswani et al., 2017.

## Getting Started

1. Open Claude Code in this directory:
   cd attention-is-all-you-need/bible/
   claude
2. Start learning â€” just chat naturally:
   explore              # see the knowledge graph
   teach me <concept>   # start a lesson
   give me a challenge  # get a quiz
   done                 # mark complete, move on

## Sharing Progress with Friends

1. Create your learner branch:
   git checkout -b learner/your-name
2. Commit progress as you learn:
   git add .learner/
   git commit -m "Progress update"
   git push origin learner/your-name
3. Fetch friends' branches:
   git fetch --all
   friends

## Course Structure

- **Foundations: Why We Needed Transformers** (6 concepts): Background concepts that motivated the Transformer architecture
- **The Attention Revolution** (5 concepts): How attention works in the Transformer: from scaled dot-product to multi-head
- **Building the Transformer** (5 concepts): How encoder and decoder stacks are constructed from attention and feed-forward layers
- **Training the Transformer** (7 concepts): Optimization techniques, regularization, and training strategies
- **Beyond Translation: Impact and Future** (4 concepts): Understanding Transformer complexity, interpretability, and future directions
