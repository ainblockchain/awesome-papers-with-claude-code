# Attention Is All You Need Learning Path

A Claude Code-powered interactive learning path based on
"Attention Is All You Need" by Vaswani, Shazeer, Parmar, Uszkoreit, Jones, Gomez, Kaiser, Polosukhin, 2017.

## Getting Started

## Sharing Progress with Friends

1. Create your learner branch:
   git checkout -b learner/your-name
2. Commit progress as you learn:
   git add .learner/
   git commit -m "Progress update"
   git push origin learner/your-name
3. Fetch friends' branches:
   git fetch --all
   friends

## Course Structure

- **Foundations** (5 concepts): Background knowledge needed to understand the Transformer architecture
- **Core Attention Mechanisms** (5 concepts): The attention operations that form the heart of the Transformer
- **Architecture Components** (6 concepts): The building blocks that form the complete Transformer
- **Training and Optimization** (5 concepts): Techniques that make Transformer training effective
- **Applications and Impact** (2 concepts): How the Transformer achieved breakthrough results and opened new frontiers
