   cd attention-is-all-you-need/
   claude

## Sharing Progress with Friends

1. Create your learner branch:
   git checkout -b learner/your-name
2. Commit progress as you learn:
   git add .learner/
   git commit -m "Progress update"
   git push origin learner/your-name
3. Fetch friends' branches:
   git fetch --all
   friends

## Course Structure

- **Foundations** (3 concepts): Background knowledge required to understand the Transformer architecture
- **Core Transformer Architecture** (5 concepts): The fundamental components that make up the Transformer
- **Encoder and Decoder Stacks** (5 concepts): Deep dive into the encoder and decoder layer structures
- **Advanced Techniques** (9 concepts): Masking, cross-attention, and training optimizations
- **Results and Future Directions** (3 concepts): The impact of Transformers and where the field is heading
