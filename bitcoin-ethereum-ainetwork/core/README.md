# Bitcoin, Ethereum, AI Network Learning Path

A Claude Code-powered interactive learning path based on Satoshi Nakamoto's
Bitcoin Whitepaper (2008), Vitalik Buterin's Ethereum Whitepaper (2014),
and AI Network Architecture/Whitepaper (2018-2025).

## Getting Started

1. Open Claude Code in this directory:
   ```bash
   cd bitcoin-ethereum-ainetwork/core
   claude
   ```
2. Start learning — just chat naturally:
   ```
   explore              # see the knowledge graph
   teach me <concept>   # start a lesson
   give me a challenge  # get a quiz
   done                 # mark complete, move on
   ```

## Sharing Progress with Friends

1. Create your learner branch:
   ```bash
   git checkout -b learner/your-name
   ```
2. Commit progress as you learn:
   ```bash
   git add .learner/
   git commit -m "Progress update"
   git push origin learner/your-name
   ```
3. Fetch friends' branches:
   ```bash
   git fetch --all
   friends
   ```

## Course Structure

- **Bitcoin Whitepaper Essentials** (4 concepts): Core concepts from Satoshi Nakamoto's 2008 Bitcoin whitepaper — transactions, proof of work, incentives
- **Bitcoin Advanced Concepts** (3 concepts): Advanced topics including SPV, state transition systems, and Bitcoin Script limitations
- **Ethereum Whitepaper Essentials** (6 concepts): Vitalik Buterin's 2014 Ethereum whitepaper — accounts, Gas, EVM, application areas
- **AI Network Architecture** (5 concepts): AI Network's Deep Computing architecture and role system — based on 2018-2025 whitepapers
- **Frontier: Future of AI and Blockchain** (2 concepts): ML patterns and Open Resource — AI Network's cutting-edge vision
