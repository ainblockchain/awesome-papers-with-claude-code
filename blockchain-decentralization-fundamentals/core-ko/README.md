# 블록체인 탈중앙화 기초

"블록체인 탈중앙화 기초"를 기반으로 한 Claude Code 인터랙티브 학습 경로입니다.
저자: 김민현, Common Computer

## 시작하기

1. 이 디렉토리에서 Claude Code를 실행하세요:
   ```
   cd blockchain-decentralization-fundamentals/core-ko
   claude
   ```
2. 자연스럽게 대화하며 학습을 시작하세요:
   ```
   explore              # 지식 그래프 보기
   teach me <concept>   # 레슨 시작
   give me a challenge  # 퀴즈 받기
   done                 # 완료 표시, 다음으로
   ```

## 코스 구조

- **모듈 1: 인터넷과 블록체인 기초** (4개 개념): 웹의 진화(1.0→2.0→3.0), 블록체인 기초, 합의 메커니즘(PoW/PoS)을 학습합니다.
- **모듈 2: 탈중앙화와 플랫폼 경제** (4개 개념): Chris Dixon의 플랫폼 생애주기 이론, bait-and-switch 문제, Voice와 Exit 솔루션, 위키피디아 vs Encarta 사례를 학습합니다.
- **모듈 3: 확장성과 미래** (3개 개념): 블록체인 트릴레마, Layer 2 확장성 솔루션, 이더리움과 EVM, DeFi/NFT 생태계를 학습합니다.

## 통계

- 3개 코스에 걸쳐 11개 개념
- 4개 기초, 4개 중급, 3개 고급 개념
