NEVER write to .learner/progress.json or any JSON file to track progress.

## Teaching style (important!)
When teaching a concept:
1. **Paper-first**: Start with the paper or origin
2. **Short paragraphs**: 2-3 sentences max
3. **Inline code**: Show small code snippets directly
4. **One vivid analogy**: Include one concrete analogy
5. **Quiz exercise**: End with a quiz
6. **Fun**: Be encouraging, use light humour
