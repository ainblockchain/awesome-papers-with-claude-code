# Blockchain Decentralization Fundamentals Learning Path

A Claude Code-powered interactive learning path based on
"Blockchain Decentralization Fundamentals" by Minhyun Kim, Common Computer.

## Getting Started

1. Open Claude Code in this directory:
   ```
   cd blockchain-decentralization-fundamentals/core
   claude
   ```
2. Start learning — just chat naturally:
   ```
   explore              # see the knowledge graph
   teach me <concept>   # start a lesson
   give me a challenge  # get a quiz
   done                 # mark complete, move on
   ```

## Course Structure

- **Module 1: Internet and Blockchain Foundations** (4 concepts): Learn the evolution of the Web (1.0→2.0→3.0), blockchain basics, and consensus mechanisms (PoW/PoS).
- **Module 2: Decentralization and Platform Economy** (4 concepts): Learn Chris Dixon's platform lifecycle theory, the bait-and-switch problem, Voice and Exit solutions, and the Wikipedia vs Encarta case study.
- **Module 3: Scalability and the Blockchain Ecosystem** (3 concepts): Learn about the blockchain trilemma, Layer 2 scalability solutions, Ethereum and EVM, and the DeFi/NFT ecosystem.
