# Web3, Blockchain & AI Fundamentals Learning Path

A Claude Code-powered interactive learning path based on
Common Computer's Web3 OJT curriculum (2024).

## Getting Started

1. Open Claude Code in this directory:
   ```bash
   cd web3-ojt-course/
   claude
   ```
2. Start learning — just chat naturally:
   ```
   explore              # see the knowledge graph
   teach me web3        # start a lesson
   give me a challenge  # get a quiz
   done                 # mark complete, move on
   ```

## Sharing Progress with Friends

1. Your progress is recorded on-chain automatically.
2. Check community stats:
   ```
   frontier             # see who's learning what
   friends              # find fellow explorers
   ```
3. Look up a friend's progress:
   ```
   friend progress 0x...
   ```

## Course Structure

### Module 1: Web3 & Blockchain Foundations
- **Web3.0 & Protocol Understanding** — Web1→Web2→Web3 evolution, protocol economies
- **DAO 101** — Smart contracts + governance tokens + on-chain treasury
- **Bitcoin & Ethereum** — P2P electronic cash, Turing-complete smart contracts

### Module 2: Decentralization & Token Economy
- **Decentralization & Consensus** — Three axes, Byzantine fault tolerance, DID
- **Strong & Weak Technology / Metaverse** — Chris Dixon's framework, 7 metaverse essentials
- **Tokenology** — Token design spaces, M×V=P×Q, Metcalfe's law

### Module 3: NFT & Digital Ownership
- **NFTs Authenticate the World** — Type/Token distinction, legitimacy theory
- **Advanced NFT Concepts** — 1,000 True Fans, evolving NFTs, Subscribe-to-Mint
- **AI x Blockchain** — Long-tail problem, GPU economics, autonomous marketplaces

### Module 4: AI Agent Economy & The Future
- **AINFT** — Autonomous Intelligent NFTs with dynamic metadata
- **ERC-8004 Trustless Agents** — Identity, Reputation, Validation registries
- **OpenClaw** — Autonomous digital worker framework, delegated authority model
