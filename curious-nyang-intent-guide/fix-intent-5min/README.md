# 궁금하냥 인텐트 5분만에 고치기

인터랙티브 미션 기반으로 궁금하냥 챗봇의 인텐트를 직접 찾고, 분석하고, 수정하고, 검증하는 실습 코스입니다.

## Course Structure

- **Stage 1 — 고칠 인텐트 찾기**: Prod 기록을 탐색하며 개선이 필요한 인텐트를 직접 발견합니다.
- **Stage 2 — 어떻게 고칠지 결정하기**: 발견한 인텐트의 수정 방향을 설명하고, AI 에이전트의 평가를 받습니다.
- **Stage 3 — 인텐트 직접 고치기**: Google Sheet에서 인텐트를 직접 수정합니다.
- **Stage 4 — 결과 확인하기**: Dev 챗봇에서 수정 결과를 직접 테스트합니다.
