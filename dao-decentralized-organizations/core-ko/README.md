# DAO & 탈중앙화 조직

Claude Code 기반의 인터랙티브 학습 경로입니다.
김민현(Common Computer) 블록체인 강의 시리즈 및 Vitalik Buterin, Taylor Pearson, Aragon, Nick Tomaino, Jesse Walden의 글을 기반으로 합니다.

## 시작하기

1. 이 디렉토리에서 Claude Code를 실행하세요:
   ```bash
   cd dao-decentralized-organizations/core-ko
   claude
   ```

2. 자연스럽게 채팅하며 학습하세요:
   ```
   explore              # 지식 그래프 보기
   teach me <concept>   # 레슨 시작
   give me a challenge  # 퀴즈 받기
   done                 # 완료 표시, 다음으로
   ```

## 코스 구조

- **DAO 기초 개념** (4개 개념): 블록체인 기초, DAO의 정의, TheDAO 해킹 사건, 기업의 존재 이유
- **DAO 의사결정 프레임워크** (3개 개념): Convex/Concave 의사결정, DAO vs Corporation 논쟁, Pod 구조
- **탈중앙화의 실제 사례** (3개 개념): 검열저항성, Kleros 분산 법원, DAO의 15가지 적용 영역
- **조직 패러다임의 전환** (4개 개념): Organization Man, Blockchain Man, Fat Protocol 테제, 기업 없이 번성하는 비트코인
- **DAO 구축 전략** (4개 개념): 탈중앙 조직의 장벽, 단계적 탈중앙화, Howey Test, Exit to Community

## 통계

- 18개 개념, 5개 코스
