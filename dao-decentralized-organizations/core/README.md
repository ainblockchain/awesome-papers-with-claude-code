# DAO & Decentralized Organizations Learning Path

A Claude Code-powered interactive learning path.
Based on Minhyun Kim (Common Computer) Blockchain Lecture Series and writings from Vitalik Buterin, Taylor Pearson, Aragon, Nick Tomaino, and Jesse Walden.

## Getting Started

1. Open Claude Code in this directory:
   ```bash
   cd dao-decentralized-organizations/core
   claude
   ```

2. Learn through natural conversation:
   ```
   explore              # see the knowledge graph
   teach me <concept>   # start a lesson
   give me a challenge  # get a quiz
   done                 # mark complete, move on
   ```

## Course Structure

- **DAO Fundamentals** (4 concepts): Blockchain basics, DAO definition, TheDAO hack, why firms exist
- **DAO Decision-Making Frameworks** (3 concepts): Convex/Concave decisions, DAO vs Corporation debate, Pod structure
- **Real-World Decentralization Cases** (3 concepts): Censorship resistance, Kleros decentralized court, 15 DAO application areas
- **Organizational Paradigm Shift** (4 concepts): Organization Man, Blockchain Man, Fat Protocol thesis, Bitcoin without a firm
- **DAO Building Strategies** (4 concepts): Decentralized org barriers, Progressive decentralization, Howey Test, Exit to Community
