"GLM-5" by Z.ai, 2025.


## Sharing Progress with Friends

1. Create your learner branch:
   ```
   git checkout -b learner/your-name
   ```
2. Commit progress as you learn:
   ```
   git add .learner/
   git commit -m "Progress update"
   git push origin learner/your-name
   ```
3. Fetch friends' branches:
   ```
   git fetch --all
   friends
   ```

## Course Structure

- **Foundations: LLM Building Blocks** (6 concepts): Essential concepts for understanding modern large language models and their training
- **GLM-5 Architecture Deep Dive** (4 concepts): The architectural innovations that make GLM-5 possible
- **Training and Alignment** (4 concepts): How GLM-5 is trained and aligned to be helpful
- **Applications and Capabilities** (3 concepts): What GLM-5 enables and how to use it
- **Frontier: Future Directions** (3 concepts): Where GLM-5's innovations lead and open questions
