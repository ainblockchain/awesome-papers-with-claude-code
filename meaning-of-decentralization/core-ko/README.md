# 탈중앙화의 의미 학습 경로

Vitalik Buterin의 "The Meaning of Decentralization" (2017)과
Web3 정체성, 헤드리스 브랜드, 분산 합의에 관한 보충 자료를 기반으로 한
Claude Code 기반 인터랙티브 학습 경로입니다.

## 시작하기

1. 이 디렉토리에서 Claude Code 열기:
   cd meaning-of-decentralization/core-ko/
   claude
2. 자연스럽게 채팅하며 학습 시작:
   탐색              # 지식 그래프 보기
   <개념> 가르쳐줘   # 레슨 시작
   문제 내줘         # 퀴즈 받기
   완료              # 완료 표시, 다음으로

## 코스 구조

- **탈중앙화의 기초** (4개 개념): Vitalik의 3축 프레임워크 — 구조적, 정치적, 논리적 탈중앙화
- **탈중앙화가 중요한 이유** (4개 개념): 장애 허용성, 공격 저항성, 담합 저항성, 공통 원인 실패
- **Web3 정체성과 헤드리스 브랜드** (6개 개념): DID, 자기주권, PKI 트로이 목마, 헤드리스 브랜드, 서사 포크
- **분산 합의 심층 분석** (4개 개념): FLP 불가능성, 비잔틴 장애 허용, 나카모토 합의

## 통계

- 4개 코스에 걸쳐 18개 개념
- 5개 foundational, 7개 intermediate, 6개 advanced 개념
