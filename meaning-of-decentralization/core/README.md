# The Meaning of Decentralization Learning Path

A Claude Code-powered interactive learning path based on
"The Meaning of Decentralization" by Vitalik Buterin (2017),
with supplementary materials on Web3 Identity, Headless Brands, and Distributed Consensus.

## Getting Started

1. Open Claude Code in this directory:
   cd meaning-of-decentralization/core/
   claude
2. Start learning — just chat naturally:
   explore              # see the knowledge graph
   teach me <concept>   # start a lesson
   give me a challenge  # get a quiz
   done                 # mark complete, move on

## Sharing Progress with Friends

1. Create your learner branch:
   git checkout -b learner/your-name
2. Commit progress as you learn:
   git add .learner/
   git commit -m "Progress update"
   git push origin learner/your-name
3. Fetch friends' branches:
   git fetch --all
   friends

## Course Structure

- **Foundations of Decentralization** (4 concepts): Vitalik's three axes framework — architectural, political, and logical decentralization
- **Why Decentralization Matters** (4 concepts): Fault tolerance, attack resistance, collusion resistance, and common mode failure
- **Web3 Identity and Headless Brands** (6 concepts): DIDs, self-sovereignty, crypto as PKI trojan horse, headless brands, and narrative forks
- **Distributed Consensus Deep Dive** (4 concepts): FLP impossibility, Byzantine fault tolerance, and Nakamoto consensus
