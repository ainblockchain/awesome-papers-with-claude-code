# NFT와 크리에이터 이코노미 학습 경로

NFT와 블록체인 기술이 어떻게 창작자와 팬의 직접적 관계를 가능하게 하는지 탐구하는 포괄적인 학습 경로입니다. Kevin Kelly의 1000명 진정한 팬 이론부터 Web3 수익화의 현실, 시장 사이클과 플랫폼 진화까지 다룹니다.

## 시작하기

1. 이 디렉토리에서 Claude Code 열기:
   ```bash
   cd nft-creator-economy/core-ko/
   claude
   ```

2. 자연스럽게 채팅하며 학습 시작:
   ```
   explore              # 지식 그래프 보기
   teach me thousand_true_fans   # 레슨 시작
   give me a challenge  # 퀴즈 받기
   done                 # 완료 표시, 다음으로 이동
   ```

## 과정 구조

- **모듈 1: 크리에이터 이코노미 기초** (3개 개념): 직접적인 창작자 수익화의 이론과 도전과제 이해하기
- **모듈 2: NFT 경제적 장점** (3개 개념): 수수료 감소, 가격 유연성, 소유권 인센티브를 통해 NFT가 창작자 경제를 근본적으로 개선하는 방법
- **모듈 3: 창작자 도구 및 플랫폼** (5개 개념): NFT 기반 창작자 수익화를 위한 실용적인 플랫폼과 메커니즘
- **모듈 4: 시장 진화 및 도전과제** (6개 개념): Web3 진화, 시장 사이클, 디자인 철학, 약속과 현실 사이의 격차 이해하기

## 주요 주제

### 크리에이터 이코노미 기초
- Kevin Kelly의 1000명 진정한 팬 이론
- 플랫폼 중개자 문제
- NFT란 무엇이며 어떻게 작동하는가?

### 경제 메커니즘
- 지대 추구 중개자 제거
- 세밀한 가격 계층화
- 소유자로서의 사용자 (CAC 제로 성장)

### 창작자 도구
- Subscribe to Mint
- Genesis 드롭 및 컬렉터블 시리즈
- Writing NFT 및 Creator DAO

### 시장 현실
- Web1, Web2, Web3 진화
- 큐레이터 경제 비전
- 스큐어모피즘 vs 크립토 네이티브 디자인
- NFT 로열티 도전과제
- 하이프 사이클과 시장 조정
