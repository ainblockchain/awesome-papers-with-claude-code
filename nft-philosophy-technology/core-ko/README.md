## 코스 구조

- **모듈 1: 철학적 기초** (3개 개념): Type vs Token 구분, Donald Davidson의 Anomalous Monism, 원목에서 모나리자까지의 대체가능성 스펙트럼 탐구
- **모듈 2: NFT 기술 표준** (4개 개념): ERC-20 (대체가능), ERC-721 (대체불가), ERC-1155 (반대체), 블록체인 표준 계층 심층 분석
- **모듈 3: 시장 역학과 경제학** (4개 개념): NFT 시장 사이클 (2021년 붐, 2023년 붕괴, 2026년 성숙), 크리에이터 로열티 (깨진 약속), Web3 게이밍 NFT, 주류 브랜드 채택 분석
- **모듈 4: 사회적 계약과 가치** (4개 개념): 사회적 계약으로서의 NFT, Vitalik의 정당성 개념, 토큰화 속성, NFT 상호운용성 이해
- **모듈 5: 미래 응용** (3개 개념): DAO-NFT 교차점, NFT-DeFi 융합, NFT가 세상을 인증하는 비전 탐구

- **철학**: Type-Token 구분, Anomalous Monism, 희소 자원으로서의 정당성
- **기술**: ERC-20/721/1155 표준, 블록체인 표준 계층, 상호운용성
- **경제학**: 시장 사이클, 크리에이터 로열티, 게이밍 NFT, 브랜드 채택
- **응용**: DAO, DeFi 통합, 미래 비전
