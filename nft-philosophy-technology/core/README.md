# NFT Philosophy and Technology Learning Path

A Claude Code-powered interactive learning course exploring the philosophical foundations of NFTs through Token Identity Theory and Donald Davidson's Anomalous Monism, understanding technical implementation via ERC-20/721/1155 standards, analyzing market cycles through a16z State of Crypto 2023, and discovering how Vitalik Buterin's concept of legitimacy explains NFT value.

## Getting Started

1. Open Claude Code in this directory:
   ```bash
   cd nft-philosophy-technology/core
   claude
   ```
2. Start learning — just chat naturally:
   ```
   explore              # see the knowledge graph
   teach me <concept>   # start a lesson
   give me a challenge  # get a quiz
   done                 # mark complete, move on
   ```

## Course Structure

- **Module 1: Philosophical Foundations** (3 concepts): Explore Type vs Token distinction, Donald Davidson's Anomalous Monism, and the fungibility spectrum from lumber to Mona Lisa
- **Module 2: NFT Technical Standards** (4 concepts): Deep dive into ERC-20 (fungible), ERC-721 (non-fungible), ERC-1155 (semi-fungible), and the blockchain standards layer
- **Module 3: Market Dynamics and Economics** (4 concepts): Analyze NFT market cycles (2021 boom, 2023 crash, 2026 maturation), creator royalties (the broken promise), Web3 gaming NFTs, and mainstream brand adoption
- **Module 4: Social Contracts and Value** (4 concepts): Understand NFTs as social contracts, Vitalik's legitimacy concept, tokenization properties, and NFT interoperability
- **Module 5: Future Applications** (3 concepts): Explore DAO-NFT intersection, NFT-DeFi fusion, and the vision of NFTs authenticating the world
