# 강한 기술과 약한 기술 학습 경로

Chris Dixon의 "Strong and Weak Technologies" (2019),
a16z crypto의 "7 Essential Ingredients of a Metaverse" (2022),
Maggie Hsu의 "Go-to-Market in Web3" (2022),
John Palmer의 "Scissor Labels" (2021)을 기반으로 한
Claude Code 기반 인터랙티브 학습 경로입니다.

## 시작하기

1. 이 디렉토리에서 Claude Code 열기:
   cd strong-weak-technologies/core-ko/
   claude
2. 자연스럽게 채팅하며 학습 시작:
   탐색              # 지식 그래프 보기
   <개념> 가르쳐줘   # 레슨 시작
   문제 내줘         # 퀴즈 받기
   완료              # 완료 표시, 다음으로

## 코스 구조

- **강한 기술 vs 약한 기술 프레임워크** (3개 개념): 세상을 재편하는 기술과 세상에 적응하는 기술을 구분하는 Chris Dixon의 기초 프레임워크
- **메타버스의 7가지 필수 요소** (7개 개념): 오픈 메타버스를 위한 a16z 프레임워크 — 탈중앙화, 재산권, 정체성, 컴포저빌리티, 개방성, 커뮤니티 소유권, 사회적 몰입
- **Web3의 시장 진출 전략** (5개 개념): 토큰이 콜드 스타트 문제를 해결하는 방법, 이해관계자 다양성, DeFi/소셜/게임 DAO별 GTM 전략
- **가위 레이블과 내러티브 전쟁** (3개 개념): Web3와 그 너머에서 모호한 트렌드 레이블이 어떻게 내러티브 통제의 전장이 되는지 이해

## 통계

- 4개 코스에 걸쳐 18개 개념
- 5개 foundational, 10개 intermediate, 3개 advanced 개념
