# Strong and Weak Technologies Learning Path

A Claude Code-powered interactive learning path based on
"Strong and Weak Technologies" by Chris Dixon (2019),
"7 Essential Ingredients of a Metaverse" by a16z crypto (2022),
"Go-to-Market in Web3" by Maggie Hsu (2022),
and "Scissor Labels" by John Palmer (2021).

## Getting Started

1. Open Claude Code in this directory:
   cd strong-weak-technologies/core/
   claude
2. Start learning — just chat naturally:
   explore              # see the knowledge graph
   teach me <concept>   # start a lesson
   give me a challenge  # get a quiz
   done                 # mark complete, move on

## Sharing Progress with Friends

1. Create your learner branch:
   git checkout -b learner/your-name
2. Commit progress as you learn:
   git add .learner/
   git commit -m "Progress update"
   git push origin learner/your-name
3. Fetch friends' branches:
   git fetch --all
   friends

## Course Structure

- **Strong vs Weak Technologies Framework** (3 concepts): Chris Dixon's foundational framework distinguishing technology pairs that reshape vs adapt to the world
- **Seven Essential Ingredients of a Metaverse** (7 concepts): The a16z framework for open metaverse — decentralization, property rights, identity, composability, openness, community ownership, and social immersion
- **Go-to-Market in Web3** (5 concepts): How tokens solve cold start problems, stakeholder diversity, and DAO-specific GTM strategies in DeFi, social, and gaming
- **Scissor Labels and Narrative Wars** (3 concepts): Understanding how ambiguous trend labels become battlegrounds for narrative control in Web3 and beyond
