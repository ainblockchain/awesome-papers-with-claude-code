## Sharing Progress with Friends

1. Create your learner branch:
   git checkout -b learner/your-name
2. Commit progress as you learn:
   git add .learner/
   git commit -m "Progress update"
   git push origin learner/your-name
3. Fetch friends' branches:
   git fetch --all
   friends

## Course Structure

- **Foundations: Transformer Architecture** (5 concepts): Core building blocks of the Llama 3 architecture including dense Transformer design, GQA, RoPE, SwiGLU, and tokenizer
- **Pre-training at Scale** (6 concepts): Data curation, scaling laws, model sizes, context extension, 4D parallelism, and infrastructure
- **Post-training Alignment** (4 concepts): SFT, reward modeling, DPO, and safety alignment techniques
- **Advanced Capabilities** (7 concepts): Synthetic data, code generation, tool use, factuality, vision, speech, and Llama Guard
- **Impact and Future Directions** (2 concepts): Benchmark performance and open release philosophy
