# 토큰 표준과 크립토 경제학 학습 경로

토큰 표준(ERC-20, ERC-721), Chris Dixon의 "Crypto Tokens" (2017)의 돌파구 통찰, a16z의 Tokenology 프레임워크 (2023), 그리고 Metcalfe의 법칙을 사용한 크립토 가치 평가 방법을 다루는 포괄적인 학습 경로입니다.

## 원본 자료

- **ERC-20 Token Standard** (Fabian Vogelsteller, 2015) — ethereum.org
- **ERC-721 NFT Standard** (William Entriken et al., 2018) — ethereum.org
- **Crypto Tokens: A Breakthrough in Open Network Design** (Chris Dixon, 2017) — Medium
- **Tokenology: Moving Beyond Tokenomics** (Guy Wuollet, 2023) — a16zcrypto
- **Valuing Bitcoin and Ethereum with Metcalfe's Law** (Jacob Franek, 2018) — Medium

## 강의 구조

**모듈 1: 토큰 기초와 표준** (6개 개념)
- 대체 가능 vs 대체 불가능 토큰
- ERC-20 표준과 토큰 손실 문제
- ERC-721 NFT 표준
- CryptoKitties 사례 연구

**모듈 2: 오픈 네트워크 원시 단위로서의 크립토 토큰** (7개 개념)
- 인터넷 중앙화 문제
- 비트코인의 이중 혁신
- 이더리움과 스마트 컨트랙트
- 토큰 기반 거버넌스와 자금 조달
- 인센티브 정렬과 토큰 분배
- 오픈 네트워크 운동

**모듈 3: 토크놀로지 - 다차원 가치** (3개 개념)
- 디지털 원시 단위로서의 토큰 (단순한 돈이 아님)
- 토크놀로지 프레임워크: 스칼라가 아닌 벡터
- 4가지 토큰 사용 사례 (소유권, 정렬, 인센티브, 접근)

**모듈 4: 크립토 가치 평가 방법** (3개 개념)
- 네트워크 가치 평가를 위한 Metcalfe의 법칙
- NVT 비율 (Network Value-to-Transactions)
- PMR 비율 (Price-to-Metcalfe)

## 통계

- 4개 모듈에 걸친 **18개 개념**
- 4개 기초, 8개 중급, 5개 고급, 1개 최전선 개념
- 개념 의존성을 보여주는 **22개 엣지**
- 2015-2026년 시간 범위와 Then & Now 분석

## 핵심 주제

1. **표준이 생태계를 가능하게 함**: ERC-20과 ERC-721이 상호운용성을 만들어, 지갑, 거래소, DApp이 수천 개의 토큰을 지원하게 함
2. **토큰이 오픈 네트워크 문제를 해결**: Chris Dixon의 통찰로 토큰이 오픈 네트워크의 운영 자금 조달과 인센티브 정렬을 가능하게 함
3. **다차원 가치**: 토큰은 단순한 경제학을 넘어 사회적, 평판적, 소유권 가치를 표현
4. **네트워크 효과 & 가치 평가**: Metcalfe의 법칙이 크립토에 적용되어 NVT와 PMR 지표를 통한 가치 평가 가능
5. **과대광고 vs 인프라**: 기술(ERC-721, 토큰 모델)은 투기 거품이 터져도 지속됨
