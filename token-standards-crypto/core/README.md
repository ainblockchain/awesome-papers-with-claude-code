# Token Standards and Crypto Economics Learning Path

A comprehensive learning path covering token standards (ERC-20, ERC-721), the breakthrough insights from Chris Dixon's "Crypto Tokens" (2017), a16z's Tokenology framework (2023), and crypto valuation methods using Metcalfe's Law.

- **ERC-721 NFT Standard** (William Entriken et al., 2018) — ethereum.org

- **Tokenology: Moving Beyond Tokenomics** (Guy Wuollet, 2023) — a16zcrypto
- **Valuing Bitcoin and Ethereum with Metcalfe's Law** (Jacob Franek, 2018) — Medium

## Key Themes

1. **Standards Enable Ecosystems**: ERC-20 and ERC-721 created interoperability, enabling wallets, exchanges, and DApps to support thousands of tokens
2. **Tokens Solve Open Network Problems**: Chris Dixon's insight that tokens enable open networks to fund operations and align incentives
3. **Multidimensional Value**: Tokens express social, reputational, and ownership value beyond mere economics
4. **Network Effects & Valuation**: Metcalfe's Law applies to crypto, enabling valuation via NVT and PMR metrics
5. **Hype vs Infrastructure**: Technology (ERC-721, token models) persists even when speculative bubbles burst

## Historical Context

The course traces token evolution from:
- 2015: ERC-20 standard proposed
- 2017: CryptoKitties launches, Chris Dixon's token breakthrough article
- 2018: ERC-721 standardized, crypto valuation methods proposed
- 2021: NFT boom, tokens as digital primitive concept
- 2023: Tokenology framework, beyond pure economics
- 2026: Market matured, utility over speculation
